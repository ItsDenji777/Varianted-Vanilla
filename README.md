###  Adding variants to mobs that need some attention! </p>

---

**Varianted Vanilla** aims to improve the design of mobs. and add extra variants for different biome types! inspired heavily by the "Spring to Life" update.


<details>
<summary>Current Mobs in the Pack</summary>

- Spider
- Pillager
- Witch
- Fox
- Goat
- Piglin

</details>

<p align=center> ⚠︎ This pack is in an early-state so it might and will be polished and updated in the near future ⚠︎</p>
